# Mac 初始化

## Homebrew 安装

  https://docs.brew.sh/Installation

```bash
  /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
```


## 安装常用软件
```bash
  brew install wget
  brew install nodejs
  brew install tree
  brew install jq
  brew install hugo
  brew install gh
  brew install golangci-lint

  # readlink
  brew install coreutils
```
